Visualization of time series using pandas,matplotlib and seaborn is done in this project.The dataset used is Occupancy detection dataset
which consists of 6 attributes and one output variable that indicates 1 if a person is present in the room ,0 otherwise. The attributes are date,temperature in Celsius,Relative Humidity in %,Light in Lux,CO2 in ppm,Humidity Ratio that is derived quantity from temperature and relative humidity, in kgwater-vapor/kg-air,Occupancy.The visualizations done here are line plots,bar charts,histograms,density plots,heatmaps,scatter plots,boxplots.From the obtained visualizations, data can be explored and various conclusions can be drawn.
